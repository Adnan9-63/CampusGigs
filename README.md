# CampusGigs
A decentralized student economy platform built to streamline peer-to-peer campus tasks, featuring AI-powered gig generation and integrated wallet simulation.
# Made by team DragonDevZ
## 🚀 Key Features
- **AI-Powered Gig Descriptions:** Uses Prompt Engineering to automatically generate professional job descriptions and pricing suggestions.
- **Student Wallet:** Simulated UPI-integrated wallet for earnings and payments.
- **Real-time Negotiation:** Built-in chat system for price negotiation.
